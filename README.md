# No-show-dashboard
A data analysis project exploring patterns in patient no-shows for medical appointments.

## Overview
This project analyzes patient appointment no-show data to identify trends and factors that contribute to missed appointments. The findings are visualized through an interactive Tableau dashboard.

### Dataset
Uses the Kaggle Medical Appointment No-Shows dataset (KaggleV2-May-2016.csv), which contains over 110,000 medical appointments in Brazil with information on patient demographics, appointment details, and whether the patient showed up for the appointment.

### Tools 
* SQL — data cleaning and exploratory analysis
* Tableau — interactive dashboard and visualizations

### Key Questions Explored
* What patient demographics are associated with higher no-show rates?
* Does the gap between scheduling and appointment date affect no-shows?
* Do chronic health issues affect no-show rates?
